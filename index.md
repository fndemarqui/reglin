# reglin

Funções para análise e conjuntos de dados dos exemplos e exercícios
utilizados na disciplina de Análise de Regressão Linear (EST035).

## Installation

O pacote **reglin** pode ser instalado da seguinte:

``` r
install.packages('reglin', repos = c('https://fndemarqui.r-universe.dev', 'https://cloud.r-project.org'))
```
